


Foram aplicadas algumas heurísticas de usabilidade para melhorar a interação. O programa mostra mensagens de status como "Conectando ao Banco Central" e "Calculando taxas", evitando que o usuário pense que o sistema travou. Também foi implementado tratamento de erros para impedir falhas caso valores inválidos sejam digitados, exibindo uma mensagem clara ao usuário.

Além disso, utilizei cores e organização no layout do terminal para deixar o sistema mais limpo e fácil de entender. O resultado final é apresentado de forma destacada e com duas casas decimais, melhorando a leitura.
